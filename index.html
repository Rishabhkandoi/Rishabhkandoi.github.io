About me

I am a dedicated Data Engineer & Data Scientist with a Master's degree in Data Science from the University of Rochester, New York. With over 3 years of industry experience in Data Engineering, I bring a strong foundation in professional ethics and interpersonal skills to any project.

My expertise spans a wide range of technologies, including Python, R, SQL, AWS, Databricks, Snowflake, Kafka, and more. I excel in data preprocessing, data cleaning, ETL, and data visualization, ensuring that your data is transformed into actionable insights.

Projects

  • Trauma Detection ML Model


    Built an ensemble ML model classifying trauma levels of patients with 90% accuracy, significantly improving on manual classification. Used statistical tests to analyze model performance across demographics.

  • Crime Rate Prediction


    Analyzed demographics, Twitter and Google searches to predict crime rates across US cities with high accuracy. Identified high crime categories needing awareness.

  • Chat Text Segmenter using NLP


    This tool segment group chat into several conversations and each conversation represents a different subject. We have used the knowledge of Natural Language Processing and Neural Networks. As part of this project, we implemented the hierarchical Bayesian unsupervised topic segmentation model technique from a research paper, and simulated it over the Slack Chat Dataset.

  • Spam Mass Detection


    A Semester 6 project under Big Data course. It filters the web pages links on the basis of in degree and out degree and thus helps to detect which pages are defined as spam and which are good ones. The programming language used is Hadoop and HDFS, and impemented in a cluster.

  • Twitter Hashtag Analysis


    A Semester 7 project under Capstone Project. It provides graphical as well as tabular output regarding popularity of hashtag on twitter using Twitter API in Python and extract profile of users contributing the most , along with various other visual analytics on a locally hosted website. The website also includes login module.

  • NetMap Network Analyzer


    Project under Programming skills course. It is a tool that uses JPCAP to analyze network and its properties by extracting information directly from the system. The interface used here for programming is Eclipse and language is Java.

  • IOT Project - Automobile Balancer


    A Semester 6 project under audit course of IOT. It focusses on the method to prevent toppling effect in automobiles and thus the solution is provided using combination of Hardware & Software, alongwith ThingSpeak as server in backend.

  • TCP Congestion Control


    Project under Computer Networks course. It has simulation and documentation for TCP Congestion control, done via NS2 as interface. All outputs are analyzed via xgraph and compared with our technique in order to reduce the congestion in a better way possible.

  • Courier Database Management


    Handling logistics and queries in a courier database system is a complex task and requires undulating attention. However it is integral to any e-commerce business or mail deliveries. So i chose the topic and created database and entity relation diagram depicting complete courier scenario. It was the Database Management Course Project.

  • PharmaConnect


    Output of Smart India Hackathon 2018 Grand Finale. An Android App that gives a common platform to integrate all the pharmaceutical groups together under one umbrella. For a non-expert, this app also provides many other features like Drugs_wiki, PharmaLocator, etc. For industry related people, it has features like Surveys, Policies, News Feed, etc.

  • Consonant


    An Android App that enables a set of people to build a knowledge repository on a specific topic or point of interest. The app has dashboard divided on the basis of interest groups, making it clean and simple to manage. People can share their state of interest, like or comment on other's posts, save their preferences to view posts based on that, etc.

  • R&D Project - VANET


    Research & Development Project done during Semester 6, focusses upon an energy-efficient approach towards network intelligence in cooperative communication in vehicular environment. It discusses the importance of using SDN along with some simulations and algorithms to prevent attacks from hackers and prvide efficient communication.

    A Semester 6 project under Entrepreneurship course. The motto was to think of a real life problem and try to build a solution around it by properly formulating the execution of complete business plan. It included brainstorming on financial forecast, creating deadlines, industry analysis, etc.

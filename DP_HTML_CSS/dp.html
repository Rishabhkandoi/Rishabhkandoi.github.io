Projects

  • Food ordering Android App

    An Android App built during internship in Jun-July 2018, that provides functionality of ordering food online. Features implemented in this app are carting, food outlets along with description of each outlet like address, map, menu, etc., gallery, login fuctionality, etc.

  • PharmaConnect

    Output of Smart India Hackathon 2018 Grand Finale. An Android App that gives a common platform to integrate all the pharmaceutical groups together under one umbrella. For a non-expert, this app also provides many other features like Drugs_wiki, PharmaLocator, etc. For industry related people, it has features like Surveys, Policies, News Feed, etc.

  • Consonant

    An Android App that enables a set of people to build a knowledge repository on a specific topic or point of interest. The app has dashboard divided on the basis of interest groups, making it clean and simple to manage. People can share their state of interest, like or comment on other's posts, save their preferences to view posts based on that, etc.

  • R&D Project - VANET

    Research & Development Project done during Semester 6, focusses upon an energy-efficient approach towards network intelligence in cooperative communication in vehicular environment. It discusses the importance of using SDN along with some simulations and algorithms to prevent attacks from hackers and prvide efficient communication.

  • IOT Project - Automobile Balancer

    A Semester 6 project under audit course of IOT. It focusses on the method to prevent toppling effect in automobiles and thus the solution is provided using combination of Hardware & Software, alongwith ThingSpeak as server in backend.

  • Spam Mass Detection

    A Semester 6 project under Big Data course. It filters the web pages links on the basis of in degree and out degree and thus helps to detect which pages are defined as spam and which are good ones. The programming language used is Hadoop and HDFS, and impemented in a cluster.
